# ペルソナレビュー指示（Agent tool用）

このファイルは `/article` の Step 4 で Agent tool（subagent）に渡すプロンプトテンプレート。

---

## Agent tool 実行時の指示

以下のプロンプトを Agent tool の `prompt` パラメータに渡す。
`{...}` 部分は実行時に実データで置換すること。

```
あなたは以下のペルソナになりきって、記事をレビューしてください。

【ペルソナ】
{step2で選定したペルソナの全セクション（基本情報・状況・悩み・課題・AIツールとの関係・読後に期待するゴール・刺さるコンテンツの傾向）}

【トーン＆マナー基準】
{00_config/concept/tone_manner.md の内容}

【レビュー対象記事】
{step3_writing.md の本文}

【レビュー観点】
以下の5つの観点で記事を評価し、それぞれ「○（問題なし）/ △（改善余地あり）/ ×（要修正）」と具体的なコメントを返してください。

1. 共感度：冒頭で「自分のための記事だ」と感じられるか
2. 理解度：専門用語や前提知識なしで内容を理解できるか
3. 納得度：主張に根拠・具体例があり、納得できるか
4. 行動度：読後に何をすればよいか明確か
5. 信頼度：煽りや誇張がなく、信頼できる文章か

【出力フォーマット】

## {ペルソナ名}のレビュー

| 観点 | 評価 | コメント |
|------|------|----------|
| 共感度 | ○/△/× | |
| 理解度 | ○/△/× | |
| 納得度 | ○/△/× | |
| 行動度 | ○/△/× | |
| 信頼度 | ○/△/× | |

### 総評
（3行以内で全体の印象をまとめる）

### 改善提案（優先度順）
1.
2.
3.

### 合格判定
- ×の数: ?/5
- △の数: ?/5
- 判定: 合格 / 再修正
（合格基準: ×がゼロ かつ △が2件以下）
```

---

## 実行パターン

### Agent tool の呼び出し方

```
Agent tool:
  subagent_type: general-purpose
  prompt: （上記テンプレートを実データで埋めたもの）
  description: "ペルソナレビュー ラウンドN"
```

### 不合格時の修正フロー

1. Agent tool の結果から ×/△ の指摘を抽出
2. **Edit** tool で `step3_writing.md` の該当箇所を修正
3. 修正内容を `step4_review.md`（または round2/round3）の「修正対応」に記録
4. 再度 Agent tool でレビュー実行（最大3ラウンド）

### 3ラウンド到達時

- 残課題をまとめてユーザーに提示
- ユーザーの判断を仰ぎ、Step 4.5 に進むか追加修正するか決める
