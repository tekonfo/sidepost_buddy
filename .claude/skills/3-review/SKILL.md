# ペルソナレビュー単体実行

`/article` の Step 4（ペルソナレビュー）を単体で実行するショートカット。
既に書き上がった記事に対してペルソナレビューだけを行いたい場合に使う。

## いつ使うか

- `/article` ワークフロー外で既存記事をレビューしたいとき
- 記事の修正後に再レビューだけ実行したいとき
- 異なるペルソナで追加レビューしたいとき

---

## 起動時の処理

以下を**並列で Read**:
- 引数の記事ファイル
- `01_strategy/03_target/persona.md`（ペルソナ定義）
- `00_config/concept/tone_manner.md`（トーン＆マナー）
- `.claude/skills/3-article/persona_review_prompt.md`（Agent tool 用プロンプト）

---

## 実行手順

1. **AskUserQuestion** でレビュー対象ペルソナを選択（引数未指定の場合）
2. `persona_review_prompt.md` のテンプレートに実データを埋め込み
3. **Agent tool** でペルソナレビューを実行:
   ```
   Agent tool:
     subagent_type: general-purpose
     prompt: （テンプレートを実データで埋めたもの）
     description: "ペルソナレビュー"
   ```
4. 5観点の結果を提示（○/△/× + 合格判定）

---

## 合格基準

1. **×（要修正）がゼロ**: レビューで × が1つもないこと
2. **△が2件以下**: 5観点中、△ が2件以下であること

---

## レビュー観点

1. **共感度**: 冒頭で「自分のための記事だ」と感じられるか
2. **理解度**: 専門用語や前提知識なしで理解できるか
3. **納得度**: 主張に根拠・具体例があり納得できるか
4. **行動度**: 読後に何をすればよいか明確か
5. **信頼度**: 煽りや誇張がなく信頼できる文章か
