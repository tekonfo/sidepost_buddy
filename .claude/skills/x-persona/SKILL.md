# ペルソナ会話シミュレーション

ペルソナになりきった Agent tool を使い、コンテンツ案に対する読者反応をシミュレーションする。

## いつ使うか

- 記事テーマの妥当性を検証したいとき
- タイトル案を比較したいとき
- 記事の下書きに対する反応を確認したいとき
- 有料記事の購入意欲をテストしたいとき
- ペルソナの悩みを深掘りしてコンテンツのヒントを得たいとき

---

## 起動時の処理

1. **Read** `01_strategy/03_target/persona.md` — ペルソナ定義を読み込み
2. 引数でペルソナ番号が指定されている場合はそのペルソナを選択
3. 未指定の場合は **AskUserQuestion** でペルソナを選択させる
4. **Read** [roleplay_prompt.md](roleplay_prompt.md) で Agent tool 用指示を読み取り
5. **Agent tool** でペルソナロールプレイを実行

---

## 実行方法

### Agent tool の呼び出し

```
Agent tool:
  subagent_type: general-purpose
  prompt: （roleplay_prompt.md のテンプレートを実データで埋めたもの）
  description: "ペルソナロールプレイ {ペルソナ名}"
```

### 使い方の例

**コンテンツ検証**: 記事の下書きや企画を見せて反応を確認
> この記事のタイトル案を3つ見せるので、どれが一番気になるか正直に教えてください。

**有料記事の購入意欲テスト**:
> こういう有料noteを980円で出そうと思っているんですが、買いますか？

**悩みの深掘り**:
> 最近一番困っていることを、もう少し詳しく聞いてもいいですか？

---

## 終了時の出力

ロールプレイ終了後、以下の振り返りを出力:

- **刺さったポイント**: ペルソナが前のめりになった話題・表現
- **刺さらなかったポイント**: ピンとこなかった話題・表現とその理由
- **発見した課題・ニーズ**: 会話の中で新たに見えた悩みや欲求
- **コンテンツへの示唆**: この会話から得られた記事企画・改善のヒント
