# Chapter 1: はじめに

## SidePost Buddy とは

SidePost Buddy は、**note 記事を AI と協働で作成するためのワークフローフレームワーク**です。

記事のネタ出しから執筆、公開後の振り返りまでを一貫した仕組みで管理し、Claude Code のスラッシュコマンド（スキル）を通じて各作業を進めます。

## 何ができるのか

- 中期的なコンテンツ戦略の策定
- テーマの発想・評価・選定
- テンプレートに沿った記事作成（素材メモ → リサーチ → 設計 → 執筆 → レビュー → 公開準備）
- ペルソナ（想定読者）によるレビューシミュレーション
- スライド画像の自動生成
- スクリーンショットのプライバシー保護加工
- 公開後のパフォーマンス分析

## 誰のためのツールか

- note で副業・情報発信をしている（または始めたい）個人
- 記事作成の型がなく、毎回ゼロから悩んでいる人
- AI を活用して効率よく質の高い記事を書きたい人

## 基本的な考え方

SidePost Buddy には 2 つの重要な原則があります。

### 1. ユーザーの体験が記事の核になる

AI が記事の「ネタ」を作ることはありません。記事の素材（体験・事実・数字）はすべてユーザー本人が提供します。AI はその素材を整理・構成・文章化する役割です。

### 2. 5 フェーズで回す

記事を 1 本書いて終わりではなく、**Strategy → Planning → Prep → Writing → Management** の 5 フェーズを繰り返すことで、コンテンツ活動全体を改善していきます。

## 前提条件

| 項目 | 必要なもの |
|------|-----------|
| ツール | Claude Code（CLI版） |
| プラットフォーム | macOS / Linux |
| 知識 | Markdown の基本（見出し・リスト程度） |
| アカウント | note のアカウント（記事公開時） |
