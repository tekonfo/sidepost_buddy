# SidePost Buddy 初心者ガイド

SidePost Buddy を使い始めるための入門ガイドです。
各チャプターを順に読むことで、セットアップから記事公開・振り返りまでの全体像を把握できます。

## 目次

| # | チャプター | 内容 |
|---|-----------|------|
| 1 | [はじめに](./01_introduction.md) | SidePost Buddy とは何か、前提条件 |
| 2 | [全体像を理解する](./02_overview.md) | 5フェーズのライフサイクルとディレクトリ構成 |
| 3 | [初期セットアップ](./03_setup.md) | ペルソナ・ブランド・トーンの初期設定 |
| 4 | [Strategy フェーズ](./04_strategy.md) | 中期計画・カレンダー・KPI の策定 |
| 5 | [Planning フェーズ](./05_ideation.md) | テーマの発想・評価・確定 |
| 6 | [Writing フェーズ](./06_writing.md) | Step 0〜5 の記事作成ワークフロー |
| 7 | [Management フェーズ](./07_analysis.md) | 公開後の振り返りと戦略フィードバック |
| 8 | [便利機能ガイド](./08_utilities.md) | ペルソナ会話・スライド・スクリーンショット等 |
| 9 | [Tips & よくある質問](./09_tips_faq.md) | ベストプラクティスと FAQ |

## 読み方のおすすめ

- **まず全体像を知りたい** → Chapter 1〜2 を読む
- **すぐに記事を書きたい** → Chapter 3（セットアップ）→ Chapter 6（Writing）へ
- **計画から始めたい** → Chapter 3 → Chapter 4（Strategy）→ Chapter 5（Planning）と順に進む
- **特定の機能を知りたい** → Chapter 8（便利機能ガイド）を参照

## 前提

- Claude Code がインストール済みであること
- プロジェクトディレクトリ（`402_side_post_buddy/`）で Claude Code を起動していること
