# Step 5: 公開準備

## 入稿チェックリスト

### メタ情報
- [ ] タイトルが適切か（32文字以内推奨）
- [ ] 公開日時を設定したか

### 本文の最終確認
- [ ] 見出しタグ（H2、H3など）が正しく設定されているか
- [ ] 画像に代替テキスト（alt）を設定したか
- [ ] CTAを設置したか（必要な場合）

---

## バナー画像チェック
### アイキャッチ画像
- [ ] アイキャッチ画像を用意したか
- [ ] 画像内のテキストが読みやすいか
- [ ] SNSシェア時の表示を確認したか

---

## 公開後の対応

### 公開直後
- [ ] 公開URLにアクセスして表示確認
- [ ] SNSでシェア（必要な場合）
- [ ] 関係者に共有（必要な場合）

### 公開後のフォロー
- [ ] Search Console でインデックス登録をリクエスト
- [ ] アクセス解析の確認（公開1日後、1週間後など）
- [ ] 必要に応じて記事を改善
